# Implementation Plan: Kolla Meeting Rebuild

## Overview

Rebuild Kolla Meeting với React 18 + Vite + Tailwind CSS (frontend), Spring Boot 3.2 (backend), Gipformer FastAPI (ASR), Jitsi Meet self-hosted (video), MySQL + Redis (data). Thứ tự: Infrastructure → Backend Core → Gipformer → Frontend Core → Meeting Room → Transcription → Minutes → Admin → Tests.
